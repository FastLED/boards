Search anything

Type a vendor name, product name, VID, or VID:PID. Case and punctuation (0x, :, spaces, dashes) are ignored — 303a:1001, 303A1001, and 0x303a 1001 all match the same product.

Vendor lookup (by VID)

Enter a 4-hex VID (e.g. 303a) or a name fragment (e.g. arduino).

Product lookup (by VID:PID or name)

Enter a VID:PID like 16c0:0483 or a name fragment like esp32.

Board definitions

Type a board name (or board id, mcu, or vendor). Click View JSON on any result to inspect the full board definition; the ↗ source link goes to the upstream repo.
